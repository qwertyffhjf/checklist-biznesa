# Чек-лист «Здоровье бизнеса» — Инструкция по запуску

## Шаг 1 — Получить бесплатный ключ Groq

1. Открыть сайт: https://console.groq.com
2. Нажать «Sign Up» → зарегистрироваться (можно через Google)
3. Перейти в раздел «API Keys»
4. Нажать «Create API Key» → скопировать ключ (начинается с `gsk_...`)
5. Сохрани ключ — он понадобится на Шаге 3

---

## Шаг 2 — Залить код на GitHub

1. Открыть https://github.com → зарегистрироваться если нет аккаунта
2. Нажать «New repository» (зелёная кнопка)
3. Назвать репозиторий, например: `checklist-app`
4. Нажать «Create repository»
5. На странице репозитория нажать «uploading an existing file»
6. Перетащить ВСЕ файлы из папки `checklist-app` (сохраняя структуру папок)
7. Нажать «Commit changes»

---

## Шаг 3 — Задеплоить на Vercel

1. Открыть https://vercel.com → нажать «Sign Up» → войти через GitHub
2. Нажать «Add New Project»
3. Найти репозиторий `checklist-app` → нажать «Import»
4. Vercel сам определит Vite — нажать «Deploy»
5. После деплоя открыть «Settings» → «Environment Variables»
6. Добавить переменную:
   - Name: `GROQ_API_KEY`
   - Value: вставить ключ из Шага 1 (`gsk_...`)
   - Нажать «Save»
7. Перейти в «Deployments» → нажать «Redeploy» (чтобы переменная применилась)

Готово! Vercel выдаст ссылку вида `https://checklist-app-xxx.vercel.app`

---

## Как работает ИИ-анализ

- Пользователь проходит анкету → отправляет результаты
- Запрос идёт на `/api/analyze` (серверная функция Vercel)
- Функция достаёт ключ из переменных окружения и вызывает Groq API
- Groq возвращает ответ **стримингом** — текст появляется по мере генерации
- Ключ никогда не виден пользователю и не хранится в коде

## Модель ИИ

Используется `llama-3.3-70b-versatile` от Groq.
- Бесплатный лимит: 14 400 запросов в день
- Скорость ответа: ~3–7 секунд (стриминг)
